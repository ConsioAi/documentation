Consio is an AI-powered voice calling platform built specifically for e-commerce brands. Whether you want to recover abandoned carts, follow up on recent orders, handle post-purchase support, or re-engage lapsed customers, Consio lets you deploy intelligent voice agents that hold natural conversations — and ties every call back to real revenue. A clean REST API means you can pull your call and order data into any BI stack, data warehouse, or custom dashboard with minimal effort.

## Who is this for?

Consio is designed for two main audiences:

- **E-commerce brands** — merchants who want to use AI voice calling to drive more revenue, reduce churn, and automate customer outreach without building a calling infrastructure from scratch.
- **BI and data teams** — analysts and engineers who need to pull call records and order attribution data into a data warehouse, reporting tool, or custom dashboard to understand campaign performance and revenue impact.

If you manage outbound sales or support for an online store, or if you need programmatic access to call and order data, Consio is built for you.

## What can you do with the API?

The Consio public API currently exposes two endpoints. Both require a valid API key and your workspace slug.

| Endpoint | Description |
|---|---|
| `GET /v1/{workspace_slug}/calls` | Retrieve a paginated list of all calls in your workspace, including metadata such as direction, duration, outcome, and timestamps. |
| `GET /v1/{workspace_slug}/attributed-orders` | Retrieve a paginated list of orders that have been attributed to a Consio call or campaign, including order value and attribution details. |

The base URL for all requests is `https://api-martin.consio.dev/public`.
